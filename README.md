# 团队饮料选择系统

一个简单的团队开会饮料选择页面，让团队成员可以方便地选择自己想要的饮料。

## 功能特点

- **饮料展示**：展示多种饮料选项，包含图片、名称和简要说明
- **用户选择**：用户输入姓名并选择饮料
- **数据记录**：自动记录所有选择（存储在浏览器本地）
- **管理后台**：查看所有选择记录，支持导出和邮件发送

## 使用说明

### 用户端
1. 访问主页面
2. 输入你的姓名
3. 点击选择你想要的饮料
4. 点击"提交选择"按钮

### 管理员端
1. 访问 `admin.html` 页面
2. 查看所有选择记录和统计信息
3. 可以：
   - 导出 CSV 文件
   - 复制邮件内容（手动发送到 bamboolin808@gmail.com）
   - 清空数据

## 技术说明

- 纯前端实现，使用 HTML + CSS + JavaScript
- 数据存储在浏览器 localStorage
- 响应式设计，支持移动端访问

## 关于邮件发送

由于纯前端页面无法直接发送邮件，管理员需要：
1. 在管理页面点击"复制邮件内容"
2. 手动将内容粘贴到邮件客户端
3. 发送到 bamboolin808@gmail.com

或者使用"导出 CSV"功能，将数据导出为 Excel 可读的格式。
